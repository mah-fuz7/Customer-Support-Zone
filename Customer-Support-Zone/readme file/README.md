

 -1.What is JSX, and why is it used?
  JSX is a syntax extension for JavaScript that allows writing UI elements in a HTML-like format inside React, making code more readable and easier to manage.

 - 2.What is the difference between State and Props?
 State is internal, mutable data managed by a component, while props are external, read-only data passed from parent to child components.

 -3. What is the useState hook, and how does it work?
`useState` is a React Hook that allows functional components to create and manage state and trigger re-renders when the state updates.

 -4. How can you share state between components in React?
 State is shared by lifting it up to a common parent component and passing it down through props.
 
 -5. How is event handling done in React?
Event handling is done using camelCase event names and passing functions as event handlers, with React managing events through a synthetic event system.
